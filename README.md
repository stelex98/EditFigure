# EditFigure
 реализовал иерархию классов, представляющих плоские геометрические фигуры. Каждая фигура производит следующие операции:
нахождение площади и периметра, центра масс (для многоугольников используется триангуляция);
изменяет координаты центра масс (при этом вся фигура сдвигается на столько же, на сколько сдвинут ее центр);
управлениет параметрами, специфическими для конкретных типов фигур (координаты вершин многоугольника, радиусы эллипса и т. п.);
осуществляет базове преобразованя: перемещение, поворот относительно точки, масштабирование относительно точки; при этом фигура  изменяться плавно в течение заданного промежутка времени и на каждый такт изменения генерировать событие (TTimer);
возможность рисования фигуры на заданной канве (TCanvas).
 Каждый класс помещён  в отдельный модуль (unit).
